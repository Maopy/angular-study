# 如何阅读

这本书是写给具有以下特质的程序员：

- 想要学习 AngularJS 的
- 已经知道 AngularJS，想要进一步学习其内部运行机制的
- 想要了解如何构建一个大型 JavaScript 应用框架的

AngularJS 并不是一个小型框架。
它做了方方面面的事情，还有很多需要掌握的新概念。
它的代码库也很庞大，有3.5万行 JavaScript 代码。
尽管这些新概念和如此之多的代码为你构建自己的应用提供了强大的工具，但是随之而来的是一个陡峭的学习曲线。

我非常讨厌使用我不太理解的技术。
因为通常，这种情况只能使我的代码恰巧能够运行，而这种运行，不是因为你真正理解了它的作用，而是你经历了大量的实验和试错。
这样写出来的代码是很难被维护和调试的。
因为你无法解释你遇到的问题。
你只能使代码看上去是能够运行的。
